いろいろな確率分布のパラメータをいじくるアプリ
=====================

## どんなアプリ？

Shiny製, 
いろいろな確率分布のカタチを見ることができるWebアプリです。  
パラメータをいじくって、確率分布のカタチがどのように変わるのか観察しましょう。  
 **左メニュー** からお好きな確率分布を選んでください。
